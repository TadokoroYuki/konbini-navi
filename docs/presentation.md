## スライド 3: ソリューション（30秒）

### コンビニ食でも栄養バランスは取れる

1. **食事バランスの簡易チェック**
   - 今日の栄養摂取を円グラフで一目で把握
   - 不足・適量・過剰をカラーで表示

2. **コンビニ商品の栄養可視化**
   - 商品を検索して記録するだけ
   - カロリー・タンパク質・脂質・炭水化物を自動計算

3. **不足を補うおすすめ商品提案**
   - 「タンパク質が足りません → サラダチキンはどうですか？」
   - 今の栄養状態に応じたパーソナル提案

## スライド 4: アーキテクチャ（30秒）

### 技術構成

```
┌─────────────┐     ┌───────────────┐     ┌──────────────────┐     ┌────────────┐
│   Expo       │────▶│  API Gateway  │────▶│  Go Microservices│────▶│ PostgreSQL │
│ (React Native)│     │               │     │  (Lambda)        │     │            │
└─────────────┘     └───────────────┘     └──────────────────┘     └────────────┘
```

- **フロントエンド**: Expo (React Native) + Expo Router v4
- **バックエンド**: Go マイクロサービス on AWS Lambda
- **サービス間通信**: gRPC
- **リポジトリ構成**: Turborepo モノレポ
- **API仕様**: OpenAPI 3.0 を Single Source of Truth
